Instruction:

Click empty block to add a Rover

"Rover facing" is default to N

Click the rover in map to highlight, or put rover id in "Rover id", then put LMR combination in "Moves" to move rover with delay

Use Arrow Keys to control highlighted rover

Notice:

Map of size (x, y) will have grid of size (x + 1, y + 1). This is because based on the given input tests, rovers need to either trespass land blocks (0, y) and (5, y) for a map size of 5 x 5.

Therefore, for this case, a map with size 5 x 5 will have bottom left (0, 0) and top right (5, 5)
